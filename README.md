# VOLTARC — Redefine Motion

Site vitrine premium pour véhicules électriques (motos & scooters).

## Déploiement GitHub Pages

1. Pusher ce repo sur GitHub
2. Aller dans **Settings → Pages**
3. Source : **Deploy from a branch**
4. Branch : `main` / `root`
5. Save → le site sera live sur `https://votre-username.github.io/voltarc-site/`

## Structure

```
voltarc-site/
├── index.html          ← Page principale
├── assets/
│   └── img/

```

## Stack

- HTML / CSS / JS vanilla (aucune dépendance)
- Google Fonts (Orbitron, Rajdhani, Space Mono)
- Animations CSS + IntersectionObserver
